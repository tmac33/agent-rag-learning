# Agent + RAG Learning Path

16-week structured learning plan for Agent creation, orchestration, and RAG systems.

**Schedule:** 1 hour/night  
**Goal:** Production-grade Agentic RAG system + certification  
**Stack:** Go (RAG services) + Python (agent orchestration)  
**Philosophy:** Claude writes all boilerplate/Python code. You focus on concepts, architecture, and Go.

## Progress

- [ ] Phase 1: Foundations (Week 1-3)
- [ ] Phase 2: Advanced RAG (Week 4-6)
- [ ] Phase 3: Agent Orchestration (Week 7-10)
- [ ] Phase 4: Engineering + Capstone (Week 11-14)
- [ ] Phase 5: Certification (Week 15-16)

## Structure

```
phase1-foundations/       → RAG basics, vector DB, tool use
phase2-advanced-rag/     → HyDE, Self-RAG, Graph RAG
phase3-agent-orchestration/ → LangGraph, multi-agent, agentic RAG
phase4-engineering/      → Evaluation, production deploy, capstone project
phase5-certification/    → Exam prep (LangChain Academy + AWS)
```

## Certifications Target

1. LangChain Academy (free, during Phase 3)
2. AWS AI Practitioner or AWS ML Specialty (Phase 5)
